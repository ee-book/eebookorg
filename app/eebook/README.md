# EE-Book

[中文](./README.md) | [English](./README_en.md)   

[EE-Book](https://github.com/knarfeh/EE-Book) 是一个命令行程序，它可以从网络上爬取内容制作成EPub格式电子书。  

[网页版](http://ee-book.org)

## 用法

获得帮助信息:  

```bash
$ python ee-book -h
```

举个例子:  

```bash
$ python ee-book -u jianshu.com/users/b1dd2b2c87a8/latest_articles
```

稍等片刻, 你就可以得到电子书了:
